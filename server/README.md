## DOOS Rental - Backend API (MVP)

This is a simple Express + SQLite backend for:

- Vehicles
- Customers
- Bookings (with overlap/availability checks)
- Basic admin login (demo-grade)

### Run

From repo root:

- `npm run dev:server`

Env (optional):

- `PORT` (default `3001`)
- `ADMIN_PASSWORD` (default `admin`)
- `DATABASE_PATH` (default `server/data.sqlite`)
